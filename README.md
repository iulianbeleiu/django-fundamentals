# Django fundamentals

#### Create and activate virtual environment
```
Create virtual environment
    python3 -m venv env

Activate virtual env:
     . env/bin/activate
     \
```

#### Install Django
```
- activate virtual env
- run: pip install django
```

#### Create Django project
```
django-admin startproject tictactoe

Run server:
    cd tictactoe
    python manage.py runserver
```
